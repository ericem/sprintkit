#######################
SprintKit, Open is Fun!
#######################

Welcome to the documentation home for the SprintKit Python API. SprintKit makes
building mobile powered Python apps super easy and lots of fun! So what exactly
can you do with SprintKit? You can send text messages to a Sprint phone, find
out if a phone is connected to the Sprint network, determine where a phone is
currently located, and receive notifications when a phone enters a geographic
to name just a few.

Well if this sounds like fun and you too would like to add mobile capabilities 
to your next Python application, then read on!
